# 📚 SeaBook

> Android-приложение для покупки, продажи и обмена книг между пользователями.

SeaBook — это мобильный маркетплейс, разработанный на Java под Android. Пользователи могут публиковать объявления о продаже книг, общаться во встроенном чате, оформлять сделки и управлять своим профилем.

---

## ✨ Возможности

- 🔐 Регистрация и авторизация пользователей
- 👤 Личный профиль с фотографией
- 📖 Добавление объявлений о продаже книг
- 🔍 Просмотр каталога книг
- 💬 Встроенный чат между продавцом и покупателем
- 🤝 Управление активными и завершёнными сделками
- 🖼 Загрузка изображений книг
- ❤️ Удобный пользовательский интерфейс

## 🚀 Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/yourusername/SeaBook.git
```

### 2. Открыть проект

Откройте папку проекта в **Android Studio**.

### 3. Подключить Firebase

Проект использует:

- Firebase Authentication
- Firebase Realtime Database
- Firebase Storage

Добавьте свой файл `google-services.json` в папку приложения.

### 4. Запустить

Подключите Android-устройство или эмулятор и нажмите **Run ▶**.
